O que é programar

o ato de programar - - captura de dados, qual dado é precisso ser capturado para funcionar - - fluxo de eventos, possiveis coisas que aconteceram que levaram o programa por caminhos - - ordem logica, O programa tem que seguir uma ordem passo a passo, não embaralhada, tem que ser ordenado - - tarefas para gerar o resultado, se o programa está chegando no resultado esperado, exemplo , pedir uma encomenda você comprar algo pela internet e ela chegar a té sua casa, se inscrever em uma academia, preparar uma comida - Como programar - - algoritimo, seria o passo a passo para resolver o problema em questão - - Lógica de programação , seria a execução do algoritmo em uma seguencia funcional - - Linguagem, seria a tradução da linguagem humana para a das maquinas onde vai ter interpetadores que faram isso. - Por que programar? - - Encontrar soluções para ganhar dinheiro, onde resoveria os problemas e cobraria por isso, mas tem que ter o entendimento do problema para que seja possivel resolvelo em código de programação - O que é um programa? - - Em um conjunto de Intruções, que um dispositivo eletronico faz para resolver determinado problema
